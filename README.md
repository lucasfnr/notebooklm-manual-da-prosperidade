# Conselheiro Financeiro Educacional — NotebookLM

Este projeto utiliza o NotebookLM como ferramenta de Inteligência Artificial Generativa aplicada à educação financeira. O objetivo é organizar, sintetizar e explicar conceitos fundamentais relacionados à saúde financeira, planejamento mensal e decisões financeiras de curto prazo. Todo o conteúdo gerado pelo notebook é baseado exclusivamente em fontes teóricas, sem risco de alucinação. O NotebookLM desenvolvido neste projeto pode ser acessado no link abaixo:

https://notebooklm.google.com/notebook/8a18770a-5276-4b5c-a333-d513dddcbc3f


## Objetivo do Projeto

> Criar um sistema de IA que funcione como um guia educacional, capaz de:
>
> - Explicar conceitos financeiros de forma clara  
> - Organizar informações complexas  
> - Gerar resumos e materiais didáticos  
> - Evitar gastos impulsivos

## Funcionalidades do Notebook
>
> - Resumos conceituais;
> - Mapas mentais;
> - Relatórios explicativos;
> - Cartões didáticos;
> - Testes conceituais;
> - Infográficos;
> - Resumo em vídeo. Para teste do notebook, é recomendado o uso de algumas das opções acima.

## Fontes

> As respostas do NotebookLM são geradas exclusivamente a partir das fontes carregadas. Foram anexados:
> 
> - materiais de educação financeira;
> - artigos acadêmicos;
> - relatórios institucionais
> - comandos com contexto para melhor funcionamento da IA.

## Tecnologias Utilizadas
>
> - NotebookLM;
> - Inteligência Artificial Generativa.

## Demonstração do Funcionamento
>
> - Interações Individuais: Exemplos de prompts focados em dúvidas comportamentais, como compras por impulso.
> - Resumo: Explicações textuais estruturadas.
> - Resumo em Áudio: Resumo gerado em formato de áudio para consumo rápido.
> - Vídeo: Demonstrações visuais de conceitos de saúde financeira.
> - Avaliações e Testes: Questionários de múltipla escolha gerados automaticamente para fixação de conteúdo.

> Para uma experiência detalhada com capturas de tela, áudios e vídeos, acesse a documentação completa em: [/demonstracao.]](./demonstracao).

 ## Observação

> Este projeto tem finalidade educacional e acadêmica e não substitui aconselhamento financeiro profissional.
